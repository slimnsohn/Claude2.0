Control

Refresh the dataset from here — no terminal needed. "Refresh" pulls the latest markets from all venues, detects rule changes, and rebuilds the export (the same job that runs automatically each morning).

Parsing rules and scoring divergence are not run here — they use Claude and depend on human review, so you run those yourself (python -m parse.rule_parser, then review).

The control server must be running (start.bat / launch.vbs starts it). The token is local-only — change RESMAP_CONTROL_TOKEN before exposing it.
