ResMap

Prediction-market resolution intelligence — a dataset that knows how markets settle, and where lookalike markets across Polymarket, Kalshi & Gemini quietly resolve differently.

Get started → How it works Live dashboard
The one idea. Anyone can store "market X trades at 42¢." ResMap stores under exactly what conditions X resolves YES, from what authoritative source, by what cutoff, and how that differs from the lookalike market on another venue. That answer is expensive to compile, compounds over time, and is the entire product.

Why it exists

A naive arb scanner sees "France wins the World Cup" at 20¢ on one venue and 24¢ on another and calls it free money. But two markets on the same real-world event can resolve differently — a different settlement source, a 6pm vs 11:59pm cutoff, a draw that voids one market and resolves the other NO, or ">50%" vs "≥50.0%" on the boundary. ResMap flags which lookalike pairs are false friends — the traps — before you treat them as the same market.

60-second taste

python -m ingest.run                    # pull all 3 venues, detect rule changes
python -m parse.rule_parser --limit 20  # LLM-extract settlement rules (you review them)
python -m parse.equivalence             # score cross-venue divergence
python -m uvicorn tool.api.main:app --port 8077   # serve it; then open the dashboard
