How It Works

The widget reads page content automatically and sends it as context to Gemini. Try asking:

  • "What's the conversion rate?"
  • "Which metrics are declining?"
  • "How was this page built?"

Integration

Add to any page with one line:

<script src="path/to/chat-widget.js"></script>
